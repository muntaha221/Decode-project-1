# TechWays - Business Landing Page

## Overview

TechWays is a modern and responsive business landing page developed using HTML, CSS, and JavaScript. The project focuses on creating a clean user interface, responsive layouts, and interactive user experience.

## Features

- Responsive Navigation Bar
- Hero Section
- About Section
- Services Section
- Features Section
- Testimonials
- Contact Form
- Scroll-to-Top Button
- Mobile Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript
